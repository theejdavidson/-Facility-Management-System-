# facility_management_system

```
Facility Management System

@authors: Ethan Davidson, Ethan Mohan, Luke Dosen on final 'java design pattern implementation'

A project for Objects, Frameworks & Patterns (LUC COMP 400-level course)
to design a scaleable system for managing facilities
containing multiple buildings and rooms within them.

Our Database Schema can be viewed at docs/schema/facility.schema.sql
and a more detailed overview of use cases and associated functionalities
can be viewed at docs/use_cases.md
```
